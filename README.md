BLOXORZ Game Search
---

Python implementation of BFS, DFS and A* search for first level of Bloxorz game.

```
#### A\* Search
```
$ python3 ./bloxorz.py

or

$ python3 ./bloxorz.py -s a-star
```
#### DFS / BFS search
```
$ python3 ./bloxorz.py -s bfs

$ python3 ./bloxorz.py -s dfs
```
Open the game and run code to auto play !
